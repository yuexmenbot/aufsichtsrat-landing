# Sebastian Müller - Aufsichtsrat & Beirat Landing Page

Eine professionelle Landing Page für Aufsichtsrats- und Beiratsmandate im deutschen Mittelstand.

## Generationen

- **Generation 0** (`index.html`) - Erste Version mit starkem Digital-Fokus
- **Generation 1** (`index-gen1.html`) - Überarbeitete Version basierend auf ICP-Feedback

## Generation 1 Verbesserungen

### Hauptänderungen basierend auf virtuellen ICP-Tests:

1. **Reduzierter Digital-Fokus**
   - Neue Headline: "Bewährte Führungsstärke" statt "Operative Exzellenz für Ihren Aufsichtsrat"
   - Weniger Betonung von Digitalisierung, mehr operative Exzellenz

2. **Erweiterte Credibility-Indikatoren**
   - Ausbildung: MBA, Diplom-Wirtschaftsingenieur
   - Aktuelle Mandate und Board-Erfahrung
   - Testimonials von Geschäftspartnern

3. **Stärkere Governance-Betonung**
   - Corporate Governance als Kernkompetenz
   - Compliance & ESG Expertise
   - Risikomanagement-Erfahrung

4. **Verbesserte Verfügbarkeits-Information**
   - Flexibilität zwischen Singapur/Deutschland
   - Reisebereitschaft und Zeitzone-Anpassung
   - Sprachen-Kompetenz

5. **Professionellere Ansprache**
   - "Vertrauliches Erstgespräch" statt "Erstgespräch vereinbaren"
   - Formellerer Ton, aber nicht steif
   - Marktübliche Vergütungsmodell-Erwähnung

## Technische Details

- **Framework:** Vanilla HTML + Tailwind CSS (CDN)
- **Responsive:** Mobile-first Design
- **Performance:** Optimiert für schnelle Ladezeiten
- **SEO:** Deutsche Meta-Tags und semantische Struktur

## Zielgruppe

- German Mittelstand (€15M-€100M Umsatz)
- Familienunternehmen in Professionalisierung
- Traditionelle Industrien mit Transformationsbedarf
- Unternehmen vor internationaler Expansion
